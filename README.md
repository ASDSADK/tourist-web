# 张家界智慧文旅监控平台

张家界国家森林公园实时客流监控与文旅资讯看板。

## 功能

- **客流趋势图** — ECharts 24 小时入园/出园/在园三线可视化
- **实时闸机模拟** — 每 2 秒随机生成进出人数，按时段智能变化
- **文旅资讯推送** — 自动推送公告/政策/天气类新闻，支持热点标记
- **承载率预警** — 根据在园人数实时计算景区承载率并分级预警
- **响应式布局** — 适配桌面/平板/手机



或使用本地服务器:

```bash
python -m http.server 8080
# 访问 http://localhost:8080
```

## 技术栈

- ECharts 5.5 (CDN)
- 原生 HTML/CSS/JavaScript
- 无构建依赖

## 部署

可部署到 GitHub Pages / Netlify / Vercel 等任意静态托管服务。
